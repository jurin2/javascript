연산자

산술연산자

종류 기본형 설명
+ A+B 더하기
- A-B 빼기
* A*B 곱하기
/ A/B 나누기
% A%B 나머지

문자열 결합연산자

종류 기본형 설명
+ A+B 둘중 한개라도 문자열 결합연산자

대입산술연산자

종류 기본형 설명
+= A+=B A = A + B
-= A-=B A = A - B
*= A*=B A = A * B
/= A/=B A = A / B
%= A%=B A = A % B

증감연산자

종류 기본형 설명
++ ++A A = A + 1 or A += 1
++ A++ A = A + 1 or A += 1
-- --A A = A - 1 or A -= 1
-- A-- A = A - 1 or A -= 1

비교연산자

종류 기본형 설명
> A > B A가 B보다 크다(초과)
< A < B A가 B보다 작다(미만)
>= A >= B A는 B보다 크거나 같다(이상)
<= A <= B A는 B보다 작거나 같다(이하)
== A == B A는 B와 같다(자료형의 일치여부 따지지 않는다)
!= A != B A는 B와 같지 않다(자료형의 일치여부 따지지 않는다)
=== A === B A는 B와 같다(자료형까지 일치)
!== A !== B A는 B와 같지 않다(자료형까지 일치)

논리연산자

종류 기본형 설명
&& 조건1 && 조건2 조건1과 조건2과 모두 true => true반환
|| 조건1 || 조건2 조건1이나 조건2중 하나라도 true => true반환
! !(조건식) true => false, false => true

삼항연산자

종류 기본형 설명
조건 ? true : false a>b ? true : false 조건식이 참이라면 true영역 실행, 거짓이라면 false영역 실행
