산술연산자

종류 기본형 설명
+ A+B 더하기
- A-B 빼기
* A*B 곱하기
/ A/B 나누기
% A%B 나머지

문자열 결합연산자

종류 기본형 설명
+ A+B 둘중 한개라도 문자열 결합연산자

종류 기본형 설명
+ A+=B A = A + B
- A-=B A = A - B
* A*=B A = A * B
/ A/=B A = A / B
% A%=B A = A % B

증감연산자

종류 기본형 설명
++ ++A A = A + 1 or A += 1
++ A++ A = A + 1 or A += 1
-- --A A = A - 1 or A -= 1
